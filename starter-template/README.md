# MCP Server Starter Template

This is a minimal MCP server template with all dependencies pre-configured.

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

Or run directly with:

```bash
npx -y ts-node --esm src/index.ts
```

## What's Included

- `package.json` - Dependencies already configured
- `tsconfig.json` - TypeScript configuration ready
- `src/index.ts` - Minimal server skeleton with commented example
